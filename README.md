# dritorjan
Help to managing files on one of my servers

```
git clone https://github.com/SimonMiaou/dritorjan.git
cd dritorjan
bundle install
```

## config.json
```json
{
  "directories" : ["/path/to/scan"]
}
```
## database.json
```json
{
  "adapter":  "postgresql",
  "host":     "localhost",
  "username": "simon",
  "password": "",
  "database": "dritorjan"
}

```
